思念如雨，在这阳春三月还飘着雪花的季节

我如雨的思念冰冻了樱花的舒展

岁月的年轮刻满了有关你的一切

如梦如幻，如醉如痴

风裹着挂牵在妩媚的樱花下缠绵

摇曳成诗

我多想，就这样沉睡在思念你的梦里

用全世界最美的声音为你吟唱

用全世界最美的色彩为你梳妆

用全世界最美的语言为你祈祷

用全世界最美的情感为你呢喃

下一场樱花雨吧

来一场桃花劫吧

让不尽的心事化为泥

念想在365个日子里飞行

却始终飞不过那一颗心的距离

我多想，就这样沉睡在思念你的梦里

夜还是那样的夜

雨还是那样的雨

但却少了你的柔情和蜜意

一个转身沉浮了多少个旧梦

一次挥手交替了多少个春秋

几多期盼，几多眷恋

如花绽放在心头

你在岸边微笑

我在对岸瞧着

你的娇娆

水波涛涛，阻挠

考考你我的爱

到底多牢靠

我把手伸过去

你把手伸过来

两手，十指

相连，紧扣

架起连心桥

我感觉到你的心跳

我的心跳

在你的眉梢撒娇

欢笑盖过河波涛

爱桥旁边筑爱巢

小河醉听情歌谣
