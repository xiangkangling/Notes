## 常用快捷键
1. 预览markdown: Ctrl + Shift + V
2. 自动保存: File -> AutoSave
3. 切换自动换行: Alt + Z

## 常用插件
1. Vetur    
    vue代码高亮显示代码补全等
2. Chinese
    vsc中文显示
3. vscode-icons
    文件图表
4. Color Picker
    代码的颜色选择器
5. JavaScript (ES6) code snippets
    支持JavaScript  ES6 语法
6. HTML Snippets
    支持HTML5标签提示
7. HTML CSS support
    css 自动补齐
8. jQuery Code Snippets
    jq代码段提示
9. open in browser
    在浏览器中打开，安装后在左侧目录中右键点击会出现 open in browser 选项
